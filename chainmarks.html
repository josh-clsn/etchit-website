chain/it · web viewer

Your chainmarks, in the browser.

A read-only static page. It decrypts your on-chain chainmarks client-side and lists each one with the matching ant-cli download command. Nothing leaves your browser except the BlockScout HTTP query — open it from file:// for fully offline use after one fetch.

Uses your browser-extension wallet (MetaMask, Rabby, Coinbase) to sign the same chainmark-derive message the mobile app uses — the key is recreated automatically, no copy-paste. The signature authorizes nothing on-chain.
