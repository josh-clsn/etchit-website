etchit — the project

the umbrella · etchit.io

etchit isn't one app — it's an idea with two apps around it. The idea: Autonomi, a public, content-addressed peer-to-peer network where you pay once to put something on it and it's reachable forever, by anyone with the address — no server, no account, nothing anyone (including the people who build etchit) can quietly take down. etch/it is how you write to it. fetch/it is how you read it. Both are open source, non-custodial thin clients — there is no etchit server in the loop, ever.

  • Built on Autonomi; payments settle on Arbitrum One; addresses are BLAKE3 content hashes
  • Non-custodial — the apps never hold your data, keys, or account
  • Reading is free on both; you only pay (a one-time fee, to the storage peers) to write
  • Reproducible builds · fully open source · org: github.com/etchit-io
Names: the wordmarks are etch/it and fetch>it (styled slash / chevron, in copper) where you control the type; in running prose write etchit and fetchit; spoken, it's "etch it" and "fetch it". [/] is the etchit glyph; [>] is fetch/it's. Lowercase throughout — never "Etchit", never "FetchIt".

Etch it once. Fetch it forever.

etchit saves text to Autonomi and gives you a single permanent address to share it. Connect your own wallet — any WalletConnect-compatible one — pay a small one-time fee to the storage peers, and what you etched stays reachable as long as the network runs. Etches can be public, or private behind biometric auth (the lookup key never publishes; it's encrypted in the Android Keystore on your device). There's encrypted backup-and-restore, and chain/it — your key addresses encrypted and pinned to your wallet on Arbitrum, so the same wallet on any device re-syncs the list (with a web viewer, too).

  • Etch text — notes, code, long-form; public or private
  • Fetch any address on the network; etchit auto-detects what comes back
  • 6-word-passphrase encrypted backups · chain/it cross-device sync
  • Non-custodial · pay once, no subscription · WalletConnect (MetaMask, Rainbow, Trust, OKX…)
  • Open source · Android & desktop in beta · iOS on the roadmap

Paste an address. See what's there.

fetchit is a read-only viewer for Autonomi. Give it any address and it fetches the content by hash and renders it — text, Markdown, JSON, PDFs, images, audio, video, archives, and full HTML pages with working scripts (the Fetch API, WebAssembly, media, range requests — a whole multi-page site can live on the network with no DNS and no origin server). No wallet, no account: fetching is always free. It's a reader, not a host — what it pulls is kept in a private, content-addressed cache on your device, the way a browser caches a page you visited; it doesn't re-serve anyone's content or hold anything for you between sessions but that cache. Note Autonomi has no streaming: a file is reassembled from every one of its chunks, so it downloads in full before it plays — once it's down, seeking is instant from the cached copy.

  • Auto-detects what an address holds and renders it inline
  • HTML pages run with real web APIs — fetch, WebAssembly, media playback, range requests
  • Whole-file download (no progressive stream), then cached for offline replay
  • Free to read — no wallet, no account · a viewer, not a server
  • Open source · Android
